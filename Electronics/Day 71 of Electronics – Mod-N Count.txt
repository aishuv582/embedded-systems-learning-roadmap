Day 71 of Electronics – Mod-N Counters

Since you've already covered counters, today's topic is an important extension: Mod-N Counters.

What is a Mod-N Counter?

A Mod-N counter is a counter that goes through N distinct states before returning to its initial state.

Formula:
N=Number of states

Examples
Mod-2 Counter → Counts: 0, 1
Mod-4 Counter → Counts: 0, 1, 2, 3
Mod-8 Counter → Counts: 0 to 7
Mod-10 Counter (Decade Counter) → Counts: 0 to 9

Decade Counter (Mod-10)

A decade counter counts:

0000 → 0001 → 0010 → ...
...
1001 (9)

After 9, it resets to:

0000

instead of continuing to 10, 11, 12, etc.

Applications
Digital clocks
Frequency counters
Event counters
Digital displays
Stopwatch circuits
Types of Mod-N Counters
1. Asynchronous Mod-N Counter
Uses ripple counting.
Simpler design.
Slower due to propagation delay.
2. Synchronous Mod-N Counter
All flip-flops receive the same clock.
Faster operation.
Preferred in modern digital systems.
