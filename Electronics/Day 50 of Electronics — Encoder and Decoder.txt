Day 48 of Electronics — Encoder and Decoder
1. Encoder

An encoder is a combinational circuit that converts multiple input lines into a smaller number of output lines.

Basic Idea
Converts 2ⁿ inputs → n outputs
Only one input should be active at a time.
Example: 8-to-3 Encoder
8 input lines
3 output lines
Active Input	Output (ABC)
D0	000
D1	001
D2	010
D3	011
D4	100
D5	101
D6	110
D7	111
Applications
Keyboard encoding
Data compression
Interrupt handling
Digital communication systems
Advantages
Reduces number of transmission lines
Simplifies digital circuits
Efficient data conversion
Disadvantages
Works properly only when one input is active
Multiple active inputs can cause ambiguity

3. Decoder

A decoder is a combinational circuit that converts binary input data into multiple output lines.

Basic Idea
Converts n inputs → 2ⁿ outputs
Example: 3-to-8 Decoder
3 input lines
8 output lines
Input (ABC)	Active Output
000	D0
001	D1
010	D2
011	D3
100	D4
101	D5
110	D6
111	D7
Applications
Memory address decoding
Display systems
Seven-segment displays
Instruction decoding in processors
Advantages
Accurate selection of outputs
Widely used in memory systems
Easy implementation
Disadvantages
Circuit complexity increases with more outputs
Requires more hardware for larger decoders
Difference Between Encoder and Decoder
Encoder	Decoder
Converts many inputs into fewer outputs	Converts fewer inputs into many outputs
2ⁿ inputs and n outputs	n inputs and 2ⁿ outputs
Used for coding	Used for decoding
Example: Keyboard encoder	Example: Memory decoder
