Day 103 of Electronics
Topic: Ring Counter and Johnson Counter

These are special types of shift register counters used for generating timing sequences and state machines in digital systems.

1. Ring Counter

A Ring Counter is a circular shift register in which the output of the last flip-flop is connected back to the input of the first flip-flop.

Working Principle
It consists of n flip-flops.
Only one flip-flop contains '1', while all others contain '0'.
On every clock pulse, the '1' shifts to the next flip-flop.
After reaching the last flip-flop, it returns to the first.
Example (4-bit Ring Counter)

Initial State:

1000

Clock Sequence:

Clock	Output
0	1000
1	0100
2	0010
3	0001
4	1000

The pattern repeats.

Characteristics
Number of flip-flops = n
Number of states = n
Requires initialization (preset)
One-hot encoding
Advantages
Very simple design
Easy decoding
Fast operation
Glitch-free outputs
Disadvantages
Not efficient
Uses many flip-flops
Only n states from n flip-flops
Applications
Timing signal generation
Sequence generation
LED chasers
Traffic light controllers
Digital control systems
2. Johnson Counter (Twisted Ring Counter)

A Johnson Counter is a modified ring counter where the complement of the last flip-flop output is connected to the first flip-flop input.

Working Principle
Consists of n flip-flops.
Inverted output of the last stage is fed back.
Produces 2n unique states.
Example (4-bit Johnson Counter)

Initial State:

0000

Clock Sequence

Clock	Output
0	0000
1	1000
2	1100
3	1110
4	1111
5	0111
6	0011
7	0001
8	0000

The sequence repeats.

Ring Counter vs Johnson Counter
Feature	Ring Counter	Johnson Counter
Feedback	Direct output	Inverted output
States	n	2n
Initialization	Required	Required
Efficiency	Lower	Higher
Hardware	More for same states	Less for same states
Speed	High	High
Complexity	Simple	Slightly more complex
