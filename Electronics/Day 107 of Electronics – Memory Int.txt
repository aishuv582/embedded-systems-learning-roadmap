Day 107 of Electronics – Memory Interfacing with 8085 Microprocessor

Today you'll learn how memory chips are connected to the 8085. This is an important interview topic for embedded systems because it combines microprocessors, digital electronics, and address decoding.

1. What is Memory Interfacing?

Memory interfacing is the process of connecting external memory (ROM/RAM) to the microprocessor so it can read and write data.

The 8085 itself does not contain program memory or data memory. External memory chips must be connected.

13. Interview Questions
Q1. Why does the 8085 need external memory?

Answer: The 8085 has no internal program/data memory, so external ROM and RAM are connected for instruction execution and data storage.

Q2. What is address decoding?

Answer: It is the process of generating the appropriate chip select signal so only the intended memory chip responds to a given address.

Q3. Why is Chip Select required?

Answer: Without Chip Select, multiple memory chips could respond to the same address, causing bus conflicts and incorrect operation.

Q4. What is the maximum memory supported by the 8085?

Answer: 64 KB.

Q5. Which signal differentiates memory and I/O operations?

Answer: IO/M̅ signal.
