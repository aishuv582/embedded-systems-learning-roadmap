Day 16 – Mesh Analysis (Loop Current Method)
🔹 What is Mesh Analysis?

Mesh Analysis is a method used to find currents in a circuit using Kirchhoff’s Voltage Law.

👉 It is mainly used for planar circuits (circuits without crossing branches).

🔹 What is a Mesh?

A mesh is a loop in a circuit that does not contain any other loops inside it.

🔹 Steps to Solve Mesh Analysis
Identify all meshes in the circuit
Assign mesh currents (I₁, I₂, …)
Apply KVL to each mesh
Express voltages using Ohm’s Law (V = IR)
Solve the equations
🔹 Key Idea (Very Important)

👉 Around any loop:
Sum of voltage drops = Sum of voltage sources
