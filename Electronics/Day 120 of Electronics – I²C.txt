Day 120 of Electronics – I²C (Inter-Integrated Circuit) Protocol


Objective

Understand the I²C protocol, its architecture, communication process, and applications.

1. What is I²C?

I²C (Inter-Integrated Circuit) is a synchronous, serial communication protocol developed by Philips (now NXP). It allows multiple integrated circuits (ICs) to communicate using only two wires.

Features
Two-wire communication
Master-Slave architecture
Supports multiple masters and multiple slaves
Half-duplex communication
Address-based communication
2. I²C Lines
SDA (Serial Data Line)
Transfers data.
Bidirectional.
SCL (Serial Clock Line)
Carries clock signal generated by the master.

Both lines require pull-up resistors because they use open-drain/open-collector outputs.

6. Start Condition

A communication begins when:

SDA changes from HIGH to LOW
while SCL remains HIGH.
SCL ───────────────

SDA ────────┐_____
            START
7. Stop Condition

Communication ends when:

SDA changes LOW to HIGH
while SCL is HIGH.
8. Addressing
7-bit Address
|7-bit Address|R/W|

Most commonly used.

10-bit Address

Used when more slave devices are required.

9. Read/Write Bit
0 → Write
1 → Read
10. ACK and NACK

After every byte transferred:

Receiver sends

ACK (0) → Data received successfully
NACK (1) → No acknowledgement
11. Data Transfer

Each byte consists of

8 Data Bits
+
1 ACK Bit

MSB is transmitted first.

12. I²C Speed Modes
Mode	Speed
Standard Mode	100 kbps
Fast Mode	400 kbps
Fast Mode Plus	1 Mbps
High Speed	3.4 Mbps
13. Advantages
Only two wires needed
Supports multiple devices
Easy PCB routing
Low hardware cost
Address-based communication
14. Disadvantages
Slower than SPI
Half-duplex
Pull-up resistors required
Bus capacitance limits cable length
15. Applications
EEPROM memory
RTC (Real-Time Clock)
Temperature sensors
OLED displays
LCD modules
Accelerometers
Gyroscopes
Battery management ICs
16. SPI vs I²C
Feature	I²C	SPI
Wires	2	4+
Speed	Up to 3.4 Mbps	Much higher (10–100+ Mbps, device-dependent)
Addressing	Yes	No
Multiple Slaves	Easy	Requires separate Chip Select lines
Duplex	Half	Full
Hardware Complexity	Low	Moderate
