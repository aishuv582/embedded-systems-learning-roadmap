DEMUX in Communication

In communication systems:

Sender transmits data through one channel
DEMUX at receiver separates data into different outputs

Used in:

Telephone networks
Data communication
Networking systems
Difference Between Decoder and DEMUX
Decoder	DEMUX
No data input	Has data input
Converts code to outputs	Routes data to outputs

A DEMUX is similar to a decoder with an added data input.
