Day 77 of Electronics – Finite State Machines (FSM)

Since you've already completed counters and shift registers, the next important digital electronics topic is Finite State Machines (FSMs).

What is an FSM?

A Finite State Machine is a digital system whose output depends on:

The current state.
The input applied.

An FSM moves from one state to another based on clock pulses and input conditions.

Basic Components
States – Different conditions of the system.
Inputs – Signals that affect state transitions.
Outputs – Signals generated by the system.
State Transition Logic – Decides the next state.
Memory Element (Flip-Flops) – Stores the current state.
Types of FSM
1. Moore Machine
Output depends only on the current state.
More stable outputs.
Output changes only when the state changes.
2. Mealy Machine
Output depends on both current state and input.
Faster response.
Requires fewer states compared to Moore machines.
Applications
Traffic light controllers
Vending machines
Elevator controllers
Digital locks
Sequence detectors
Communication protocols
