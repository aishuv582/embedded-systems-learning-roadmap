Day 101: Registers
What is a Register?

A register is a group of flip-flops used to store binary data temporarily.

Each flip-flop stores 1 bit.
An n-bit register consists of n flip-flops.
Example:
4-bit register → 4 flip-flops
8-bit register → 8 flip-flops

Registers are widely used in:

Microprocessors
Microcontrollers
Digital systems
Embedded systems
Block Diagram

D1 ---->|D FF|---- Q1

        Common Clock

All flip-flops share the same clock, so all bits are stored simultaneously.

Parallel Register

A parallel register loads all bits at the same time.

Example:

Input:

1011

After one clock pulse:

Output:

1011

All bits are transferred simultaneously.

Working Principle

Suppose a 4-bit register:

Inputs

D3 D2 D1 D0
1  0  1  1

After the rising edge of the clock:

Q3 Q2 Q1 Q0

1 0 1 1

The register stores the data until:

New data is loaded, or
A reset signal clears it.
