1. What is the 8255 PPI?

The 8255 Programmable Peripheral Interface (PPI) is a programmable I/O device used to connect the 8085 microprocessor with external peripherals such as:

LEDs
Keyboards
LCDs
Switches
Printers
Sensors

It provides 24 programmable I/O pins.

2. Block Diagram

The 8255 consists of:

Data Bus Buffer
Read/Write Control Logic
Group A Control
Group B Control
Port A (8-bit)
Port B (8-bit)
Port C (8-bit)

Port C can be split into:

Upper Port C (PC7–PC4)
Lower Port C (PC3–PC0)
3. Ports
Port A
8-bit I/O port
Can operate in Modes 0, 1, and 2
Port B
8-bit I/O port
Supports Modes 0 and 1
Port C
8-bit port
Can be used as:
General-purpose I/O
Handshaking signals
Two 4-bit ports
4. Operating Modes
Mode 0 – Simple I/O
Basic input/output
No handshaking
Outputs are latched
Inputs are not latched

Applications: LEDs, switches

Mode 1 – Strobed I/O
Uses handshaking signals
Supports interrupt generation
Suitable for slower peripherals

Applications: Keyboard, printer

Mode 2 – Bidirectional Bus
Available only for Port A
Supports two-way communication
Uses handshaking signals

Applications: Data communication

5. BSR Mode (Bit Set/Reset)

Used to control individual bits of Port C without affecting the other bits.

Example:

Set PC5
Reset PC2
6. Control Word

The control word configures:

I/O mode
Input/output direction of each port
BSR operations

It is written into the Control Register.

7. Applications of 8255

Industrial automation
Motor control
