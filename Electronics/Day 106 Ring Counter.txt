Day 106: Ring Counter
What is a Ring Counter?

A Ring Counter is a type of shift register counter in which the output of the last flip-flop is connected to the input of the first flip-flop. A single logic '1' (or sometimes a single '0') circulates continuously through the flip-flops with each clock pulse.

It is also known as a One-Hot Counter because only one flip-flop is HIGH at any given time.

Block Diagram (4-bit Ring Counter)
       Clock
         │
    +----+----+----+----+
    | DFF0 | DFF1 | DFF2 | DFF3 |
    +----+----+----+----+
      │      │      │      │
      └──────►──────►──────►
              ▲            │
              └────────────┘

The output of the last flip-flop is fed back to the first flip-flop.

Initial Condition

A Ring Counter must be initialized with one flip-flop set to 1 and all others set to 0.

Example:

Q3 Q2 Q1 Q0
0  0  0  1

Without this initialization, the counter may remain in the all-zero state.

Operation

Assume the initial state is:

0001

After each clock pulse:

Clock Pulse	Output
Initial	0001
1	0010
2	0100
3	1000
4	0001
5	0010

The logic 1 moves in a circular path, creating a repeating sequence.

Number of States

For an n-bit Ring Counter:

Number of flip-flops = n
Number of valid states = n

Example:

Flip-Flops	States
4	4
5	5
8	8
Why is it called a One-Hot Counter?

Because only one output is HIGH ('1') at any time.

Example:

1000
0100
0010
0001

Each state has exactly one HIGH bit.

Advantages
Simple design
Easy to decode outputs
No complex combinational logic
Fast operation
Glitch-free decoding
Suitable for sequence control
Disadvantages
Requires more flip-flops than binary counters
Needs proper initialization
Less efficient in terms of hardware usage
Applications
Sequence generators
LED chasing/light-running displays
Traffic light controllers
Stepper motor control
Timing and control circuits
Finite State Machine (FSM) implementation
Digital control systems
Ring Counter vs Shift Register
Shift Register	Ring Counter
Stores and shifts data	Generates a repeating sequence
May not use feedback	Uses feedback from last to first flip-flop
Used for data transfer	Used for counting and sequencing
Ring Counter vs Binary Counter
Ring Counter	Binary Counter
n states for n flip-flops	2ⁿ states for n flip-flops
One output HIGH at a time	Multiple outputs may be HIGH
Easy to decode	Requires decoding logic
More hardware	Hardware efficient
