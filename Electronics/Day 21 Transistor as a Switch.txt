Day 21: Transistor as a Switch
🔹 What is a Transistor Switch?

A transistor can work like an ON/OFF switch in circuits.

ON state → Current flows (Switch closed)
OFF state → No current (Switch open)

This is widely used in:

Embedded systems
Microcontrollers
Digital circuits
🔹 Types of Transistor (Quick Recall)

We mainly use:

NPN transistor (most common)
PNP transistor

👉 Today we focus on NPN transistor as a switch

🔹 Working of NPN Transistor Switch
1. OFF State (Cut-off Region)
Base current = 0
Transistor is OFF
No collector current flows

👉 Acts like open switch

2. ON State (Saturation Region)
Base current is given
Transistor fully ON
Maximum collector current flows

👉 Acts like closed switch

🔹 Simple Circuit Idea
Base → Input signal (through resistor)
Collector → Load (LED, motor, etc.)
Emitter → Ground

🔹 Example

👉 You want to control an LED using a microcontroller:

Microcontroller gives HIGH (5V) → Transistor ON → LED glows 💡
Microcontroller gives LOW (0V) → Transistor OFF → LED OFF
🔹 Why This is Important for You (Embedded Systems)

This concept is very important because:

Microcontrollers cannot drive high current devices directly
Transistor acts as a bridge between low power and high power circuits
🔹 Real-Life Applications
Relay switching
Motor control
LED drivers
Power control circuits
