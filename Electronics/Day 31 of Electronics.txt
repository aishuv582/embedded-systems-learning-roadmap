🔹 2. Gain Bandwidth Product (GBW)

👉 Real op-amps:

Gain decreases as frequency increases

Gain×Bandwidth=Constant

Meaning:
High gain → low bandwidth
High bandwidth → low gain

👉 Very common interview question.

🔹 3. Input Offset Voltage

👉 Ideally:

Output = 0 when input = 0

👉 Practically:

Small unwanted voltage exists → causes error
🔹 4. Bias Current

👉 Small current flows into input terminals
(not exactly zero)

Effect:
Creates voltage drop → error in output
